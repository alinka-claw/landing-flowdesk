# Landing FlowDesk

**7-секционный лендинг** с hero-секцией и прайсингом. Сгенерирован Qwen, адаптирован для GitHub Pages.

## 📌 Структура
```
.
├── index.html    # HTML-разметка (7 секций)
├── style.css     # Стили (адаптив + анимации)
├── script.js     # Интерактив (меню, скролл, форма)
└── README.md      # Этот файл
```

## 🚀 Как использовать
1. **Локально**: Открой `index.html` в браузере.
2. **GitHub Pages**: Включи хостинг в настройках репо (`Settings > Pages`).

## 🎨 Дизайн
- **Тема**: FlowDesk (цифровое агентство для командной работы).
- **Цвета**: Фиолетовый (`#6c3bff`) + светлый фон.
- **Анимации**: Плавное появление при скролле, ховер-эффекты.

## 🔧 Технические детали
- **Адаптивность**: Breakpoints на 480px, 768px, 1024px.
- **JS**: Слайдер отзывов, мобильное меню, валидация формы.
- **Без библиотек**: Чистый HTML/CSS/JS.
